  <action>
    Create sam_pipeline/extract.py with one public function and several private helpers:

    Public API:
    ```python
    def run(config, logger):
        """Resolve download URL, download zip, extract .dat, return .dat path."""
    ```

    Private helpers:
    - `_resolve_api_url(config, logger)` -- builds URL from `config["portal"]["url"]` base + `config["portal"]["api_key"]` + query params `fileType=ENTITY&sensitivity=PUBLIC&frequency=MONTHLY&charset=UTF8`. Logs the URL with api_key replaced by `***`. Returns the full URL string.
    - `_download_zip(url, dest_path, logger, connect_timeout, read_timeout)` -- `requests.get(url, stream=True, timeout=(connect_timeout, read_timeout))`, calls `resp.raise_for_status()`, iterates `resp.iter_content(chunk_size=65536)`, writes chunks to dest_path, logs progress every 10MB (10 * 1024 * 1024 bytes). Logs start message with total size from Content-Length header if available, logs completion with total MB and elapsed seconds.
    - `_download_with_retry(url, dest_path, logger, max_retries, connect_timeout, read_timeout)` -- for loop 1..max_retries, calls `_download_zip`, catches `requests.exceptions.RequestException`, on non-final attempt logs warning and sleeps `2 ** attempt` seconds, on final attempt raises. On any failure, saves response body (if available) to `os.path.join(os.path.dirname(dest_path), "last_portal_page.html")` per D-07.
    - `_extract_dat(zip_path, work_dir, logger)` -- opens zip with `zipfile.ZipFile(zip_path, "r")`, finds first member ending in `.DAT` (case-insensitive via `m.upper().endswith(".DAT")`), extracts it to work_dir, returns absolute path. Raises `RuntimeError` if no .dat member found. Uses `os.path.basename(member)` joined with work_dir to avoid path traversal from zip member names containing directory separators.

    run() logic:
    1. `temp_dir = config["download"]["temp_dir"]` then `os.makedirs(temp_dir, exist_ok=True)`
    2. Check `config["portal"].get("download_url", "").strip()` -- if non-empty, use it directly (D-06), log "Using manual override URL from config"
    3. Otherwise call `_resolve_api_url(config, logger)`
    4. Derive zip filename: parse URL or default to `sam_extract.zip`. Use `os.path.join(temp_dir, zip_filename)` as dest.
    5. Call `_download_with_retry(url, zip_path, logger, max_retries=int(config["download"]["max_retries"]), connect_timeout=int(config["download"]["connect_timeout"]), read_timeout=int(config["download"]["read_timeout"]))`
    6. Call `_extract_dat(zip_path, temp_dir, logger)`
    7. Return dat_path

    Imports: `import os, time, zipfile, logging, requests`

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Pipeline -> SAM.gov API | HTTP request to external government API with API key |
| SAM.gov API -> Pipeline | Downloaded zip file from internet (untrusted content) |
| config.ini -> Pipeline | API key read from local config file |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Information Disclosure | extract.py logging | mitigate | Mask api_key value with `***` in all log output. Never log the full URL containing the raw key. Implemented in `_resolve_api_url` and `_download_with_retry`. |
| T-02-02 | Tampering | _extract_dat zip member names | mitigate | Use `os.path.basename(member)` before joining with work_dir to prevent path traversal via crafted zip member names like `../../etc/passwd`. |
| T-02-03 | Information Disclosure | config.ini contains api_key | accept | config.ini is local-only, listed in .gitignore pattern via config.ini.example convention. API key is free and scoped to public data only. Low-value target. |
| T-02-04 | Denial of Service | Large zip download fills disk | accept | SAM.gov monthly file is ~124MB, predictable size. No mitigation needed for a known-size government data file on a server with adequate disk. |
| T-02-05 | Spoofing | SSRF via configurable portal URL | mitigate | The `url` in config.ini is set by the operator, not user input. The `download_url` override is also operator-configured. No external user can influence these values. Accept as operator-trusted config. |
| T-02-06 | Information Disclosure | temp/last_portal_page.html debug dump | accept | Contains SAM.gov public page content only, no secrets. Temp directory is local. Useful for debugging portal changes. |
</threat_model>
